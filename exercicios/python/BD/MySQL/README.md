- ‚úÖ **Senha incorreta**: Verifique a senha no `config.env`
- ‚úÖ **Usu√°rio n√£o existe**: Crie o usu√°rio ou use `root`

**Solu√ß√µes:**

- ‚úÖ **Reiniciar MySQL**: `sudo systemctl restart mysql`

## Diferen√ßas entre MySQL e PostgreSQL

### Tipos de Dados
- **MySQL**: `AUTO_INCREMENT` vs PostgreSQL `SERIAL`
- **MySQL**: `INT` vs PostgreSQL `INTEGER`
- **MySQL**: `VARCHAR` vs PostgreSQL `VARCHAR` (similar)
- **MySQL**: `TIMESTAMP` vs PostgreSQL `TIMESTAMP` (similar)

### Sintaxe SQL
- **MySQL**: `AUTO_INCREMENT PRIMARY KEY`
- **PostgreSQL**: `SERIAL PRIMARY KEY`
- **MySQL**: `ENGINE=InnoDB` (opcional)
- **PostgreSQL**: N√£o usa engines

### Conex√£o
- **MySQL**: Porta padr√£o 3306
- **PostgreSQL**: Porta padr√£o 5432
- **MySQL**: `mysql.connector`
- **PostgreSQL**: `psycopg2`
